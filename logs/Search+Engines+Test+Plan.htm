Warning
This test plan should performs only on cluster with installed Data Capture.
Warning
To trigger channel by Google search engines you should to use Advanced Search or SWITCH OFF Google Instant(Live Search) because REQ-1983
Testing version 1.12.0.0 and above
Take into consideration that a new anti-fraud system is now deployed and the user is not getting more then 20 ads per minute. Test engineer should make sure he is not requesting too many ads within 1 minute especially when performing autotest. You should re-Opt-in in case you found that server stopped serving ads to you because of the anti-fraud system.
